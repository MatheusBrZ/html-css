Principais Formatações

Destaque / Ênfase

Para usar destaque usamos <strong> </strong>

Para usar ênfase usamos <em> </em>

Marcações

Para usarmos marcações, usamos <mark> </mark>

Letra Grande e Pequena

Para letras grande usaremos <big> </big> (Tag Obsoleto)

Para letras pequenas usaremos <small> </small>

Textos Deletados

Para textos deletados usaremos <del> </del>

Textos Inseridos

Para textos inseridos usamos <ins> </ins>
